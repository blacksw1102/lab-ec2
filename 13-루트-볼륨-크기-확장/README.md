## 주의사항

한번 증축한 볼륨에 대해서는 다시 크기를 축소시키는 것은 불가능하다. 볼륨 크기 확장 작업을 할 때는 반드시 사전에 스냅샷으로 백업을 해둬야 하고, 만약에 기존 볼륨에 대해서 크기를 줄여야만 하는 상황이 생긴다면 크기가 작은 새 볼륨을 생성해서 디스크 자체를 복사해줘야 하는 불편한 상황이 발생할 수 있으니 주의를 할 필요가 있다.
